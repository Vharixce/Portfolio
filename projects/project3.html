Project "Open Hiring" is a project I had to work on for a company called "IO Digital". This company wanted a website/app which uses anonymous applications; to combat preferential hiring practices and to protect the applicants privacy for as long as possible. This application was usually meant for people who face negative judgement based on physical or mental shortcomings. For these reasons, I've made the applicant's information completely anonymous by only letting the user choose between 5 preset profile pictures, only showing their account ID as a name, and only showing the result of a personality/skills test (see image 2) to the employer, who can in turn filter on these traits.

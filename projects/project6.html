Project Overview

Project 'Pose Detection' is a web application designed to recognize and train hand poses, specifically for sign language, but its able to be trained for various purposes. The application utilizes advanced pose detection algorithms to identify hand gestures and provide real-time feedback to users, ensuring a responsive and user-friendly interface.
